Informations sur le stage

Titre du stage : Développement d'une plateforme web éducative

Objectifs et missions

Ce stage permet à l'étudiant de découvrir un environnement de développement professionnel dans une entreprise innovante. Il contribue à un projet web avec une équipe agile.

Compétences visées

  • Développement web fullstack (Vue.js / Spring Boot)
  • Design de base de données relationnelles
  • Méthodes agiles (Scrum)
  • Tests unitaires et d’intégration

Conditions de stage

Gratification : 600 € / mois

Assurance : Étudiant assuré MAIF – responsabilité civile

Horaires : Lundi à vendredi, 9h → 17h

Engagements

  • Respect du règlement intérieur de l’entreprise
  • Confidentialité des informations sensibles
  • Respect du planning défini par l’encadrant
  • Remise d’un rapport à la fin du stage
